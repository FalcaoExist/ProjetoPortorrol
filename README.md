Portorrol é uma empresa especializada na comercialização e distribuição de rolamentos e componentes industriais. Este projeto tem como objetivo modernizar e automatizar os processos da empresa, unificando o backend e o frontend em uma arquitetura dockerizada.

O sistema foi desenvolvido seguindo princípios de arquitetura limpa e SOLID, garantindo escalabilidade, manutenção simplificada e documentação clara.

A aplicação é dividida em dois módulos principais:

* **Backend:** construído em Python (FastAPI), responsável pelas regras de negócio e integração com o banco de dados.
* **Frontend:** desenvolvido em React, oferecendo uma interface moderna e responsiva.

O projeto utiliza **Docker** e **docker-compose** para gerenciar os serviços de forma padronizada, facilitando a implantação e o ambiente de desenvolvimento.

**Principais objetivos:**

* Centralizar e automatizar o controle de produtos e pedidos.
* Melhorar a rastreabilidade de processos internos.
* Facilitar a manutenção e expansão futura do sistema.

**Tecnologias principais:**

* Python (FastAPI)
* React
* Docker e docker-compose
* PostgreSQL 

Este repositório representa a base do novo sistema digital da Portorrol, buscando eficiência, padronização e crescimento tecnológico.
